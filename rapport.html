Mauvaises pratiques :

  • Il vaudrait mieux écraser les styles bootstrap au lieu de mettre des !important
  • D'une manière générale, bootstrap n'est pas rentabilisé, il y a beaucoup de lignes css qui auraient pu être écrites en utilisant les commandes du framework

Dependencies :

  • Bootstrap & jquery
    • Utiliser un CDN et exploiter la possibilité que l'utilisateur ait déjà bootstrap en cache dans son navigateur (ou charger uniquement les composants dont on a besoin avec Sass)
    • Mettre à jour pour éviter les vulnérabilités
      • Télécharger la pièce jointe
  • Gmaps : Mettre simplement un iframe et économiser le temps de chargement (ou utiliser vraiment les potentialités de l'API)
  • Icônes : choisir une seule library, font awesome ou ET-line ou mieux : ne garder que les icônes qui sont utilisées sur le site

Autres modifs :

  • Choisir des url lisibles : page 2 ➔ formulaire contact**
  • Mettre des liens dans les icones de réseaux sociaux**
  • Compléter la balise title vide ➔ mot clef**
  • Minifier fichiers CSS & JS
  • Images : .bmp ➔ webp & retailler si elles sont trop grandes & compresser l'image de fond
  • Structure de titres
    • Télécharger la pièce jointe
  • Sémantique : div ➔ section, relier label aux inputs avec attribut for

Référencement

  • Eviter les annuaires**
  • Trouver des partenaires dans le même secteur**
  • Mettre un texte aguicheur dans la balise meta description**
  • Rédiger des textes plus longs, minimum 500 mots**

Crawlers :

  • Ajouter un fichier robot.txt
  • Formater l'adresse en RDFa avec schema.org
  • Ajouter un Sitemap

Ce qui améliore le référencement c'est aussi le nombre de visites donc peut-être qu'il serait bien de réfléchir à actualiser la présentation afin d'avoir un site plus attractif. Ce serait également l'occasion de créer une page blog ou news qui serait régulièrement mise à jour et d'améliorer l'UX design. La décision de conserver et rentabiliser Bootstrap où de l'abandonner complètement pour alléger le site et améliorer le temps de chargement pourrait également être prise à cette occasion. Une stratégie de partenariat avec d'autres agents dans le même secteur est à envisager, afin d'ajouter au site des liens valides et de qualité. Enfin, une fois ce travail effectué, afin de suivre l'évolution SEO, il est nécessaire d'installer google analytics
